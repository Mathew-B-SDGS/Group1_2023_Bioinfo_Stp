Understanding R-Numbers and Gene Panels

The National Genomic Test Directory serves as a cataloge of the various genomic tests availble on the NHS (in England).

Please find a link to the latest version of the National Genomic Test Directory:

NHS England

Please note version 5.1 (June 2023) is being used in this software

R-codes are unique identifiers assigned to individual genomic tests within the directory. These alphanumeric codes help streamline the search process, making it efficient for healthcare professionals and researchers to locate specific tests.

Some R-codes are associated with an NGS test which require gene panels to specify the region of interest for sequencing. These regions of interest are defined within a BED file.

PanelSearcher allows you to search for panel information using a R-code and create a BED file for the panel.

For more information, visit the Panel App, or check out this projects Github availble at STP projects
